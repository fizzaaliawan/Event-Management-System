# EventSphere

## Project Overview
EventSphere is a fully responsive web application that allows users to explore upcoming events with details like title, date, venue, and description. The platform features interactive event cards, detailed event pages with RSVP/Register functionality, and embedded Google Maps for location visualization.  

This project demonstrates modern frontend and full-stack web development skills using **React, Next.js, Node.js, TypeScript, and Tailwind CSS**.

---

## Tech Stack Used
- **Frontend:** React, Next.js, Tailwind CSS  
- **Backend:** Node.js, TypeScript  
- **Others:** Responsive Design, Modern UI/UX  

---

## Capabilities
- Browse upcoming events with key details (event name, date & time, venue, short description)  
- Filter events using a search bar  
- View detailed event pages with larger banners, full descriptions, RSVP/Register buttons  
- Interactive Google Maps for event locations  
- Contact form to submit inquiries  
- Fully responsive design for desktop, tablet, and mobile devices
